## Run it on your machine

1. Clone the project

   ```sh
   git clone https://github.com/aznamle/E-commerce-project.git
   ```

1. Install the dependencies

   ```sh
   cd E-commerce-project
   npm install
   ```

3. Start the development server

   ```sh
   npm start
   ```
